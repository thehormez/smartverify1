سياسة الخصوصية — KRS INVITE

آخر تحديث:

1) من نحن

KRS INVITE هي خدمة لإدارة الدعوات والتحقق من الحضور باستخدام الباركود/QR، وقد تتضمن إرسال رسائل عبر WhatsApp لتمكين الضيوف من استلام معلومات الدعوة.

2) البيانات التي نجمعها

  • الاسم ورقم الهاتف (لأغراض إرسال الدعوة وإدارة الحضور).
  • رموز الدعوة/الباركود المرتبطة بالضيف.
  • بيانات تشغيلية مثل حالة الإرسال والحضور (مرسلة/فشل/تم الاستخدام).

3) كيف نستخدم البيانات

  • إرسال رسائل الدعوة/التأكيد عبر WhatsApp عند الحاجة.
  • إدارة قوائم المدعوين ومتابعة حالة الدعوات والحضور.
  • تحسين جودة الخدمة وحل المشاكل الفنية.

4) مشاركة البيانات

لا نبيع بياناتك. قد تتم مشاركة البيانات فقط مع مزودي الخدمات التقنيين الضروريين لتشغيل النظام، مثل: Meta WhatsApp Cloud API لإرسال الرسائل، وGoogle Firebase/Cloud لتخزين بيانات التشغيل (مثل صور الباركود) وتشغيل الوظائف السحابية.

5) التخزين والأمان

يتم تخزين البيانات باستخدام خدمات سحابية آمنة (مثل Firebase/Cloud Storage) مع تطبيق ضوابط وصول مناسبة. ومع ذلك لا يمكن ضمان أمان 100% لأي نظام.

6) مدة الاحتفاظ

نحتفظ بالبيانات فقط للمدة اللازمة لتشغيل الفعالية وإعداد التقارير، أو حسب ما يتطلبه القانون/الجهة المنظمة.

7) حقوقك

  • طلب حذف أو تعديل بياناتك.
  • الاستفسار عن البيانات التي نحتفظ بها.

8) التواصل

للتواصل بخصوص الخصوصية: krs2.store@gmail.com

Privacy Policy — KRS INVITE

Last updated:

1) Who we are

KRS INVITE provides invitation management and attendance verification using QR/Barcodes, including optional WhatsApp messaging to deliver invitation details.

2) Data we collect

  • Name and phone number (to deliver invitations and manage attendance).
  • Invitation/QR codes associated with each guest.
  • Operational data such as delivery/attendance status (sent/failed/used).

3) How we use data

  • Send invitation/confirmation messages via WhatsApp when required.
  • Manage guest lists and track invitation/attendance status.
  • Troubleshoot issues and improve service quality.

4) Data sharing

We do not sell personal data. We may share data only with essential service providers such as Meta WhatsApp Cloud API (messaging) and Google Firebase/Cloud (hosting/storage).

5) Storage & security

Data may be stored using secure cloud services (e.g., Firebase/Cloud Storage) with appropriate access controls. No system can guarantee 100% security.

6) Retention

We retain data only as needed for event operations/reporting or as required by law.

7) Your rights

  • Request deletion or correction of your data.
  • Request information about stored data.

8) Contact

Privacy contact: krs2.store@gmail.com
